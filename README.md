# 🤖 Baloo Bot — My First Slack Bot

A Slack bot built with Node.js and Slack Bolt, hosted 24/7 on Hack Club Nest.

## What it does
- Responds to slash commands like `/dsb-ping`
- Replies to messages in Slack channels
- Runs 24/7 without needing a computer on

## Built with
- JavaScript (Node.js)
- Slack Bolt library
- Socket Mode (no public URL needed)
- Hosted on Hack Club Nest

## Commands
| Command | Description |
|---------|-------------|
| `/dsb-ping` | Check if the bot is online and see latency |

## How I built it
1. Created a Slack app at api.slack.com
2. Enabled Socket Mode and generated tokens
3. Wrote the bot code in JavaScript
4. Cloned the repo onto Hack Club Nest
5. Set it up as a systemd service to run 24/7
